Technologies

This is my first AngularJS and Bootstrap frontend. The backend is made with Java EE on a Wildfly application server.
The services are running in the cloud on two Cloud at Cost VMs. Each service is running in an own Docker container.
The security of the admin site is made with Keycloak and Java EE security.

Professional competence

The focus of this page is to inform people about the upcoming creek restructuring in Kirchberg.
The site consist of two business parts. One part is the open website with some static pages about the backgrounds and a dynamic front page with current news and status updates. This dynamic updates can be made by the owner himself over a secured admin page. He can create, update and delete entries as well as update and delete comments. New comments can be made from everyone. One does not need to have a login for the page. The entries and comments are displayed in the admin page with a live preview that the owner can see the results in real time.
