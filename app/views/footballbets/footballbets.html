Footballbets

Technologies

This is my first NodeJS project. I have used the following technologies:
  • NodeJS
  • Express
  • Jade
The application is running on Azure as a website.

Professional competence

The site displays all football games with the corresponding rates. The user can register itself and bet on these games.
The admin can set the rates and close the game by setting the winner or as a tie. The betting is automatically closed when the start time of the game is reached.
